 What is Operating System??

 -- OS is software that manages the whole computer.
 -- It work like middleman bet the user and the computer hardware.
 -- OS decide how much and when resource need to work.

 Operating System does --
 #Run the application -- vs code , chrome etc.
 #Manage memory -- RAM usage
 #MAnges files -- create/ delete files.
 #Manages hardware devices -- Keyboard, mouse , printer, etc..
 #Manges CPU -- which app get the CPU time.


 What is Kernal??
 -- Kernal is the core(heart) of Operating System.
 -- Kernal directly communicate with hardware for Operating System.

 kernal does --
  # CPU management -- which program run first.
  # Memory mangement -- Allocate RAM.
  # process Management -- Running programs.
