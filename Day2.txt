History of Node.js --

- Node.js was created in 2009 by Ryan Dahl.

- He wanted to solve the problem of slow web servers that handled many connections at the same time.

- Traditional servers created one thread per request, which consumed more memory.

- Node.js introduced NPM for managing packages and is widely used for building APIs, real-time apps, and backend services.
